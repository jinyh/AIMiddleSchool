# 高中人工智能基础教材

面向高一、高二学生的人工智能教材编写项目。

## 项目简介

本项目旨在编写符合高中课标要求的人工智能基础教材，以基础知识夯实和创新应用为核心，培养学生的AI素养和实践能力。

- **目标学段**：高一年级（基础知识）、高二年级（创新应用）
- **课标依据**：高中课标必修1（新版）及选必4
- **教材特色**：项目驱动、跨学科融合、自主可控、实践导向

详细提纲请查看：[高一AI教材提纲 v1.0](docs/高一AI教材提纲-v1.0-20260416.md)

## 教材特色

### 1. 项目驱动学习
每章设置真实情境项目，从问题出发引导学习，项目难度螺旋上升。

### 2. 跨学科融合（10-20%内容）
融合数学、物理、生物、社会学、艺术等学科知识。

### 3. 自主可控精神
每章结尾介绍国产技术和平台：
- 国产数据库（OceanBase、TiDB）
- 国产深度学习框架（飞桨PaddlePaddle、MindSpore）
- 国产大模型（文心一言、通义千问）
- 中国AI应用成果（智慧城市、智能制造）

### 4. 实践能力培养
- 使用Python作为主要编程语言
- 每章包含"动手实践"和"拓展探究"栏目
- 提供配套实验手册和在线平台

### 5. 思政元素融入
- 数据安全与国家安全
- AI伦理与社会责任
- 科技自立自强精神
- 中国智慧与创新成果

## 编写规范

### 内容要求
- 准确严谨，符合学生认知规律
- 政治方向正确，禁止特定厂商信息
- 坚持"国内为主、自主可控"原则
- 每章包含10%-20%跨学科融合知识点

### 格式要求
- 正文：小四号字体，1.2倍行距
- 篇幅：全书约160页（高一教材）
- 图文配比：每页至少一个图
- 禁止：二维码、外链接
